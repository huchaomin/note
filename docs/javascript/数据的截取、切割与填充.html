# 数据类型的截取、切割与填充

# Array

let arr = [1, 2, 3, 4, 5]
  1. slice 复制一段出来用 (不会改变原数组)
arr.slice(1) // [2, 3, 4, 5]
arr.slice(2, 3) // [3]
arr.slice(-2) // [4, 5]
arr.slice(-3, -2) // [3]
  1. splice 重新拼接组装原来的数组 (会改变原数组)
arr.splice(1, 2)
// [2, 3]
// arr => [1, 4, 5]

arr.splice(-3, 2, 7)
// [3, 4]
// arr => [1, 2, 7, 5] 		只填充一次

arr.splice(-3, 2, 7, 8)
// [3, 4]
// arr => [1, 2, 7, 8, 5]

# String

let str = 'abcdefg'
  1. slice 复制一段出来用 (不会改变原字符串)
str.slice(1)	// 'bcdefg'
str.slice(2, 3)	// 'c'
str.slice(-2)	// 'fg'
str.slice(-3, -2) // ‘e’
  1. split 切割成数组 (不会改变原字符串)
str.split()	// ['abcdefg']
str.split('')	// ['a', 'b', 'c', 'd', 'e', 'f', 'g']
str.split('c')	// ['ab', 'defg']
  1. substring(不会改变原字符串)
  • 正常情况与silce相同
  • 如果 indexStart 等于 indexEnd，substring 返回一个空字符串。
  • 如果省略 indexEnd，substring 提取字符一直到字符串末尾。
  • 如果任一参数小于 0 或为 NaN (opens new window)，则被当作 0。
  • 如果任一参数大于 stringName.length，则被当作 stringName.length。
  • 如果 indexStart 大于 indexEnd，则 substring 的执行效果就像两个参数调换了一样
str.substring(1)	// 'bcdefg'
str.substring(2, 3)	// 'c'
str.substring(-2)	// 'abcdefg'
str.substring(-3, -2) // ’‘
