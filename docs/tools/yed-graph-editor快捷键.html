# 快捷键列表

  • F2 选中节点后添加文本
  • Shift + enter 编辑文本时换行文本
  • Enter 编辑文本时，结束编辑文本
  • Shift
    • 拖动缩放节点时，保持节点的宽高比
    • 单击节点时可进行多选
    • 拖动节点时，可以按住，移除或者移入group
  • Ctrl + shift 拖动缩放节点时，以中心为缩放点，保持节点的宽高比
  • Ctrl + a 选中一个边时，按下可选中所有边（同理节点，边上的文字）
  • Ctrl + d 快速创建兄弟元素
  • Ctrl + w 快速创建子节点
