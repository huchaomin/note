# 规则码

  • ”off” or 0 - 关闭规则
  • ”warn” or 1 - 将规则视为一个警告（不会影响退出码）
  • ”error” or 2 - 将规则视为一个错误 (退出码为1)

# 使用顺序

  1. .eslintrc.js
  2. .eslintrc.yaml
  3. .eslintrc.yml
  4. .eslintrc.json
  5. .eslintrc
  6. package.json

# ignore

除了 .eslintignore 文件中的模式，ESLint总是忽略 /node_modules/* 和 /bower_components/* 中的文件。
