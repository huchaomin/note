# row-gap、column-gap和gap

# 兼容性

can-i-use-row-gap (opens new window) ie 不兼容

# 语法

  • row-gap: 行间距
  • column-gap: 列间距
  • gap: 行列间距的合并简写，语法如下
gap: <row-gap> <column-gap>;
/*如果省略了第二个值，浏览器认为第二个值等于第一个值*/
警告

三者旧语法为：grid-row-gap grid-column-gap grid-gap

#container {
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}
