# configureWebpack 方式

  1. 它可以是一个对象：和 webpack 本身配置方式是一致，该对象将会被 webpack-merge 合并入最终的 webpack 配置
configureWebpack: {
  rules: [],
  plugins:  []
}
  1. 它也可以是一个函数
configureWebpack: (config) => {
  // 例如，通过判断运行环境，设置mode
  config.mode = 'production'
}

# chainWebpack 方式

chainWebpack: (config) => {
  config.mode = 'production'
}
